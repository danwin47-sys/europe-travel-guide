🎓 學生/青年省錢攻略 (Student Money Saver)

您的旅伴中有一位學生 (或是 27 歲以下)，請務必隨身攜帶 國際學生證 (ISIC) 或護照。這趟旅程至少可以省下 €100+！

1. 交通折扣 (Transport)

  • ÖBB 火車：遺憾通知，奧地利鐵路青年卡 (Vorteilscard Jugend) 嚴格限制 26 歲以下。37 歲學生無法享有此 5 折優惠。建議提早買早鳥票 (Sparschiene) 最划算。
  • 維也納交通：學生票限制 26 歲以下。請直接購買成人票 (7-Day Ticket)。
  • 布達佩斯交通：學生票通常只認歐盟學生證，國際學生證 (ISIC) 常被拒。建議買成人票。

2. 門票折扣 (Attractions) - 37歲學生攻略

  • 維也納：
  • 美泉宮/美景宮：學生票多限制 25-26 歲以下。37 歲持有 ISIC 可能無法折價 (視現場寬容度，但要有心理準備買全票)。
  • 歌劇魅影 (Phantom)：青年票 (Jugendticket) 限制 27 歲以下。
  • *替代方案*：37 歲學生建議排當天的 "Restkarten" (剩餘票) 或 "Stehplatz" (站票)，一樣很便宜。
  • 薩爾茲堡：
  • 鹽礦：通常接受 ISIC 國際學生證 (折扣約 €3)，年齡查核較寬鬆。
  • 布達佩斯：
  • 馬提亞斯教堂/漁人堡：這是少數明確接受 ISIC 且無年齡上限的地方！請務必出示學生證，可享半價。
  • 溫泉：無學生優惠。

🎭 常見騙術與應對

1. 假警察 (Fake Police)

手法：便衣「警察」要檢查你的護照或錢包，說要查毒品/假鈔。

應對：

  • ❌ 絕對不要給護照或錢包
  • ✅ 要求一起去最近的警察局
  • ✅ 真警察會有制服和警徽，便衣警察極少查觀光客

2. 簽名黨 (Petition Scam)

手法：拿板子要你連署簽名（聾啞人士、慈善等），簽了就討錢。

應對：

  • ❌ 不要眼神接觸
  • ✅ 搖頭快步走開
  • ✅ 雙手插口袋或抱胸

3. 幸運繩 (Friendship Bracelet)

手法：有人要把繩子套在你手上，套上就要錢。

應對：

  • ❌ 不要伸手
  • ✅ 雙手插口袋或抱胸
  • ✅ 大聲說 "No, thank you!"

4. 計程車詐騙 (Taxi Scam)

手法：路邊攔車，司機繞遠路或跳錶。

應對：

  • ❌ 不要搭路邊排班計程車
  • ✅ 用 Bolt 或 Uber App 叫車
  • ✅ 上車前確認車牌和司機名字

5. 餐廳帳單詐騙

手法：帳單多加項目（麵包、水、服務費）。

應對：

  • ✅ 仔細檢查帳單每一項
  • ✅ 不確定就問 "What is this?"
  • ✅ 拍照存證

  • 匈牙利：布達佩斯自來水理論上可生飲，但管線較舊，擔心的話建議在超市買瓶裝水 (Ásványvíz)。
  • *粉紅蓋*：無氣泡水 (Still)。
  • *藍蓋/綠蓋*：氣泡水 (Sparkling/Gas)。別買錯！

2. 廁所 (Toilets)

  • 要付費：歐洲公廁幾乎都要錢 (0.5€ - 1€)。
  • 省錢撇步：
  • 火車上、餐廳用餐後必上。
  • 博物館、百貨公司通常免費。
  • 隨身準備幾個 0.5€ 硬幣。
  • 找廁所 App：Flush、Toilet Finder

3. 小費 (Tipping)

  • 餐廳：這裡不像美國強制，但有服務若是滿意的，習慣給 5% - 10%。
  • 給法：結帳時直接告訴服務生你要付的總金額。
  • 例：消費 27.5€，拿 50€ 給他並說 "Thirty (30), please."，他就會找你 20€。

4. 營業時間

  • 週日：幾乎所有商店關門（超市、藥妝店都關）
  • 超市：平日通常 8pm 關門
  • 餐廳：午餐 12:00-14:00，晚餐 18:00-22:00

🆘 緊急情境應對手冊

情境 1：迷路了怎麼辦？

1. 不要慌：找個安全的地方（咖啡廳、商店）

2. 開 Google Maps：確認目前位置

3. 問路：找年輕人或商店店員（英文通）

4. 最後手段：叫 Uber/Bolt 回飯店

情境 2：錯過火車怎麼辦？

1. 立刻去售票櫃台：說 "I missed my train"

2. 改簽：大部分車票可以改簽下一班（可能要補差價）

3. 買新票：如果改不了，買下一班的票

4. 通知飯店：如果會晚到，打電話通知飯店

情境 3：護照掉了怎麼辦？

1. 報警：去最近的警察局報案，拿報案證明

2. 聯絡代表處：

  • 駐奧地利：+43-664-345-0455
  • 駐匈牙利：+36-30-964-1601

    • 3. 準備文件：護照影本、大頭照、報案證明

    4. 申請臨時護照：代表處會協助

情境 4：生病了怎麼辦？

1. 輕症：去藥局 (Apotheke) 買藥

2. 中症：打電話給飯店，請他們推薦診所

3. 重症：打 112 叫救護車

4. 保險：記得拿收據申請保險理賠

情境 5：被偷了怎麼辦？

1. 報警：打 112 或去警察局

2. 掛失：

  • 信用卡：立刻打電話掛失
  • 手機：遠端鎖定/清除資料

    • 3. 聯絡代表處：如果護照也被偷

    4. 備用方案：

  • 護照影本存雲端
  • 信用卡掛失電話存手機
  • 緊急現金藏在不同地方

情境 6：語言不通怎麼辦？

1. Google Translate：拍照翻譯功能超好用

2. 比手畫腳：手勢 + 微笑 = 萬能

3. 找年輕人：年輕人英文通常比較好

4. 用圖片：把想要的東西拍照給對方看
